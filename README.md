==============================Install==============================

1. reference 디렉토리의 sql.zip을 자신의 DB에 import한다.

2. db.php를 열어 자기 DB에 맞는 username, password, dbname을 적는다.

2-1. 자신의 DB가 비밀번호를 사용한다면, mysql_connect 함수의 파라미터로 $password를 추가한다.
